# PyAutoLens

Final project for Computational Physics at NCSSM.

This project explores strong gravitational lensing using PyAutoLens, analyzing how light from distant galaxies is bent by the gravitational field of foreground lens galaxies.

## Notebooks

- `introduction.ipynb` - Overview of PyAutoLens features and API
- `lenses.ipynb` - Basic lensing calculations with light and mass profiles
- `interferometry.ipynb` - Interferometry analysis for submillimeter and radio observations
